# GY-NEXUS 쇼핑 쇼츠 전용 제작실 V2.1

## 접속 위치

- 관리자 메뉴: `쇼핑 쇼츠 센터`
- 경로: `/admin/shopping-shorts`
- 배포 후 예상 주소: `https://gy-nexus-zfpq.vercel.app/admin/shopping-shorts`

기존 배포 사이트에는 새 소스가 설치되고 Git에 반영된 뒤 Vercel 배포가 끝나야 표시됩니다.

## 대표 사용 순서

1. 상품 링크·상품명·확인된 설명·상품 사진을 넣고 프로젝트를 만듭니다.
2. `도우인·샤오홍슈 쇼츠 수집함`에서 한국어 또는 중국어 검색어로 각 플랫폼 검색을 엽니다.
3. 참고할 쇼츠의 공유 페이지 주소를 등록합니다.
4. 직접 촬영·판매자 제공·제휴센터 제공·사용 허가 영상은 파일로 올리고 정확한 권리 상태를 선택합니다.
5. 핵심 장면 캡처가 있으면 분석 프레임을 최대 8장 올린 뒤 `AI 장면 분석`을 누릅니다.
6. 목록에서 사용할 항목의 `AI 믹스 선택`을 켭니다.
7. `선택 소스로 AI 짜집기 설계`를 누릅니다.
8. AI가 만든 15~30초 컷 타임라인에서 `허가 원본 사용`, `새로 촬영·생성`, `새 AI 장면`을 확인합니다.
9. 상품화 준비 → 훅 선택 → 대표 품질 승인 → 음성·장면 생성 → 최종 MP4 렌더 순서로 진행합니다.
10. SRT, CapCut 가이드, CapCut 편집 JSON, 게시 패키지를 내려받습니다.

## 도우인·샤오홍슈 연결 범위

- 도우인 공식 OpenAPI는 앱 심사와 사용자 승인을 받은 계정의 영상 목록·업로드·게시 범위로 연결할 수 있습니다.
- 일반 타인 영상 전체를 상품 키워드로 검색해 자동 다운로드하는 공식 API 연결은 제공하지 않습니다.
- 샤오홍슈 공식 공개 개발 문서는 상거래 API 중심이며, 일반 공개 콘텐츠 검색·다운로드 기능으로 구현하지 않습니다.
- 그래서 현재 V2.1은 플랫폼 검색 실행 → 공유 링크 수집 → AI 분석 → 허가 영상만 최종 편집하는 방식입니다.
- 공식 앱 승인을 받은 뒤 `.env.local`에 플랫폼 앱 키를 넣으면 화면에서 준비 상태를 확인할 수 있습니다.

## 품질·권리 안전장치

- 권리 미확인 링크도 훅·촬영각도·판매 구조 분석 대상으로 선택할 수 있습니다.
- 페이지 링크와 권리 미확인 영상은 최종 원본 컷으로 사용할 수 없습니다.
- AI 믹스가 잘못 `허가 원본 사용`을 제안해도 서버가 `새로 촬영·생성`으로 자동 변경합니다.
- 최종 렌더 Worker에는 실제 업로드된 허가 영상만 전달됩니다.
- 소스·권리·분석 결과가 변경되면 이전 믹스와 콘텐츠 승인이 자동 해제됩니다.
- 상품 사실 설명, 첫 3초 훅, 정확한 SRT, 제휴 고지를 통과해야 게시 단계로 이동합니다.

## 환경변수

기존 필수 환경변수 외에 공식 앱 심사를 받은 경우에만 아래 값을 설정합니다.

```env
DOUYIN_CLIENT_KEY=
DOUYIN_CLIENT_SECRET=
DOUYIN_REDIRECT_URI=
XIAOHONGSHU_APP_KEY=
XIAOHONGSHU_APP_SECRET=
```

허가 영상 업로드에는 공개 읽기가 가능한 `CREATIVE_STORAGE_BUCKET`과 충분한 파일 크기 제한이 필요합니다. V2.1 파일 선택 한도는 500MB이며, Supabase Storage 버킷 제한이 더 작으면 버킷 설정을 먼저 높여야 합니다.
