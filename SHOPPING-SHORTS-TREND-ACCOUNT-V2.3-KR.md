# GY-NEXUS 쇼핑 쇼츠 트렌드·계정연결 V2.3

## 대표님 요청을 다시 맞춘 핵심 흐름

```text
한국어 상품명 입력
→ 중국어 간체 상품명·검색어 5~8개 자동 생성
→ 한국어를 검색에 쓰지 않고 중국어로 도우인·샤오홍슈 검색
→ 확인 가능한 인기 쇼츠를 먼저 수집
→ 인기 결과가 없으면 상품·문제·사용 상황 관련 쇼츠로 자동 재검색
→ 공개 좋아요·반복 태그처럼 확인된 인기 신호만 표시
→ 대표가 여러 카드 선택
→ AI 소스함 저장
→ 15~30초 한국형 판매 순서 재구성
```

## V2.3에서 달라진 기능

- 한국어만 입력해도 중국어 간체 상품명으로 자동 변환합니다.
- 번역이 실패하면 한국어로 대신 검색하지 않고 오류를 알려 잘못된 검색 결과를 막습니다.
- 상품명, 문제 해결, 사용 상황, 후기, 바이럴 표현을 나눈 중국어 검색어를 5~8개 보여줍니다.
- 실제 도우인·샤오홍슈 검색에는 중국어 간체 상품명과 중국어 키워드만 사용합니다.
- 인기 신호가 확인된 쇼츠를 먼저 찾고, 없으면 관련 쇼츠 검색으로 자동 전환합니다.
- 키워드 칩을 누르면 해당 중국어로 즉시 다시 검색합니다.
- 60초를 넘는 것으로 확인된 영상은 결과에서 제외합니다.
- 결과 카드를 9:16 세로형 쇼츠 비율로 표시합니다.
- 공개 좋아요 수치가 있으면 표시하고, 없으면 검색 상위 후보로만 표시합니다.
- 같은 태그가 여러 결과에서 나타나면 `반복 노출` 또는 `계정 인기 태그`로 올립니다.
- 선택한 쇼츠 카드의 대표 이미지를 AI 믹스의 시각 참고자료로 사용합니다.

## Edge 로그인 계정 연결 방식

도우인·샤오홍슈 일반 회원가입 정보만으로 Vercel 서버가 계정 검색 권한을 얻을 수는 없습니다. 비밀번호나 로그인 쿠키를 서버에 넣는 방식은 보안상 사용하지 않습니다.

V2.3은 대신 Edge 로컬 확장 프로그램을 제공합니다.

- 대표님 Edge에 이미 로그인된 도우인·샤오홍슈 세션을 그대로 사용합니다.
- 검색은 GY-NEXUS에서 시작합니다.
- 확장 프로그램이 플랫폼 검색 탭을 백그라운드에서 열고 화면에 실제 표시된 카드만 읽습니다.
- 개별 쇼츠 링크, 썸네일, 공개 좋아요 문구, 길이, 해시태그만 GY-NEXUS로 보냅니다.
- 쿠키 권한이 없으며 비밀번호·쿠키·영상 파일 주소를 읽거나 전송하지 않습니다.
- 로그인 확인이나 보안 문자가 나오면 자동 수집을 멈추고 해당 탭을 보여줍니다.

## Edge 확장 프로그램 설치

V2.3 PowerShell 설치가 완료되면 다음 폴더가 생성됩니다.

```text
C:\Users\홍영택\Documents\GY-CHINA-ACCOUNT-CONNECTOR-EDGE
```

1. Edge 주소창에 `edge://extensions`를 입력합니다.
2. 왼쪽 아래 `개발자 모드`를 켭니다.
3. `압축 풀린 항목 로드`를 누릅니다.
4. `C:\Users\홍영택\Documents\GY-CHINA-ACCOUNT-CONNECTOR-EDGE` 폴더를 선택합니다.
5. 도우인과 샤오홍슈에 각각 로그인해 둡니다.
6. GY-NEXUS 쇼핑 쇼츠 센터를 새로고침합니다.
7. `Edge 로그인 계정 연결 · 연결됨`이 표시되는지 확인합니다.

## 실제 사용 순서

1. `/admin/shopping-shorts`를 엽니다.
2. 작업할 상품 프로젝트를 선택합니다.
3. `한국어 상품명 입력`에 `손선풍기`처럼 입력합니다.
4. `중국어로 자동 번역해 찾기`를 누릅니다.
5. 중국어 상품명과 검색 키워드를 확인합니다.
6. 인기 쇼츠가 있으면 `중국어 인기 쇼츠`, 없으면 `인기 결과가 없어 찾은 관련 쇼츠`로 구분되어 표시됩니다.
7. 공개 웹 검색 카드가 먼저 표시되고, Edge 도우미가 연결되어 있으면 같은 중국어 검색어로 로그인 계정의 검색 카드가 이어서 합쳐집니다.
8. 쇼츠 카드의 썸네일, 길이, 공개 인기 신호, 원문을 확인합니다.
9. 필요한 카드 여러 개를 선택하고 `AI 믹스 준비`를 누릅니다.
10. `선택 소스로 AI 짜집기 설계`를 눌러 15~30초 판매 순서를 만듭니다.

## 인기 표시의 의미

- `화면 좋아요`: 로그인 계정 검색 화면에 공개된 좋아요 문구를 읽은 값입니다.
- `공개 좋아요`: 공개 페이지 메타데이터에 실제 값이 있을 때만 표시합니다.
- `계정 검색 상위`: 현재 대표님 계정의 네이티브 검색 결과에서 위쪽에 노출된 순서입니다.
- `반복 노출·계정 인기 태그`: 여러 검색 카드에 같은 키워드나 해시태그가 나타난 횟수입니다.
- `관련 쇼츠`: 인기 수치는 확인되지 않았지만 중국어 상품명·해결 문제·사용 상황과 가까운 결과입니다.
- 확인되지 않은 조회수·판매량·전 세계 인기 순위는 만들지 않습니다.

도우인 네이티브 검색 결과는 계정·지역·시점에 따라 달라질 수 있습니다. 공식 키워드 검색 API는 앱 권한과 키워드 등록이 필요하므로, 심사 후에는 공식 공급자로 교체할 수 있습니다.

## 최종 영상 사용 범위

검색 카드와 로그인 계정 결과는 트렌드 구조 분석용입니다. 원본 영상을 자동 다운로드하거나 워터마크를 제거하지 않습니다. 원본 장면을 최종 영상에 사용하려면 직접 촬영, 판매자 제공, 제휴센터 제공 또는 사용 허가 영상을 별도로 올려야 합니다.
