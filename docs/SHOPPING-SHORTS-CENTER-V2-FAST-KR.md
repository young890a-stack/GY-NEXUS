# GY-NEXUS 쇼핑 쇼츠 센터 V2 FAST

## 어디에서 확인하나요?

배포 후 관리자 왼쪽 메뉴의 **CONTENT → 쇼핑 쇼츠 센터**를 누릅니다.

```text
https://gy-nexus-zfpq.vercel.app/admin/shopping-shorts
```

이 주소는 새 소스가 Vercel에 배포된 뒤 열립니다. 현재 운영 사이트에 파일을 복사만 하고 배포하지 않았다면 이전 화면이 계속 보입니다.

## 대표 사용 순서

1. 인기상품·저장 상품을 불러오거나 쿠팡·Temu 상품명, 설명, 사진, 제휴 링크를 입력합니다.
2. 사진 한 장 또는 프리미엄 2~4장 모드를 선택해 프로젝트를 만듭니다.
3. 참고할 도우인·샤오홍슈·판매자 영상 URL과 권리 상태를 등록합니다.
4. 영상 핵심 장면을 캡처한 사진을 최대 8장 올리고 **AI 장면 분석**을 누릅니다.
5. 추출된 한국어·중국어 키워드 중 사용할 키워드를 선택합니다.
6. AI의 프레임별 **유지 후보 / 제거 / 새로 제작** 판단과 0.7~2.5초 믹스 순서를 확인합니다.
7. **쇼핑 쇼츠 상품화 준비**를 눌러 한국형 훅·대본·정확한 자막·썸네일·4개 플랫폼 패키지를 만듭니다.
8. 훅 하나를 선택하고 대표 품질 승인을 진행한 뒤 AI 음성과 장면 영상을 만듭니다.
9. CapCut 자료를 받거나 FFmpeg Worker로 최종 MP4를 합성합니다.
10. 완성본은 YouTube 비공개 대기열에 넣어 게시센터에서 최종 확인합니다.

## 이번 V2에 포함된 기능

- 관리자 메뉴에 독립 **쇼핑 쇼츠 센터** 추가
- 인기상품·저장 상품 및 쿠팡·Temu 수동 링크 연결
- 사진 1장 또는 2~4장 상품 쇼츠 제작
- 참고 URL과 최대 8장 분석 프레임 등록
- URL 메타데이터·프레임·메모 기반 상품명/검색 키워드 추출
- 도우인·샤오홍슈 중국어 탐색 링크 생성
- 훅, 판매 포인트, 장면 역할 분석
- 상품과 무관한 장면 제거, 유용하지만 복제하면 안 되는 장면 재제작 판단
- 0.7~2.5초 컷의 새로운 영상 믹스 순서
- 한국형 훅 3개, 대본, 정확한 SRT, AI 음성 선택
- 자막 스타일, BGM 분위기, 효과음 방향
- 정확한 한국어 글자의 썸네일 3안과 SVG 저장
- YouTube Shorts, Instagram Reels, Douyin, Xiaohongshu 패키지
- CapCut 안내서, CapCut 편집 JSON, 게시 패키지
- 상품 이미지 AI 품질검수와 Runway 비용 승인 장치
- Docker용 FFmpeg Worker: 컷 정규화·믹스·자막·음성·허가 BGM·MP4·콜백

## 빨라진 부분

- 중국어 트렌드 설계와 판매 패키지를 순차가 아니라 병렬 생성합니다.
- 같은 입력이면 검수 완료 결과를 캐시에서 즉시 불러옵니다.
- 결과를 완전히 새로 만들 때만 **전체 AI 결과 다시 생성**을 누릅니다.
- 최대 8장의 분석 프레임을 동시에 업로드합니다.
- 화면 밖의 무거운 프로젝트·장면 카드는 필요할 때 렌더링합니다.
- 일반 생성은 빠른 모델·중간 추론을 사용하고, 독립 광고 검수는 고품질 모델·높은 추론을 유지합니다.

권장 환경변수:

```text
OPENAI_FAST_MODEL=빠른-생성용-모델
OPENAI_QUALITY_MODEL=독립-품질검수용-모델
```

## FFmpeg Worker 연결

Vercel 서버리스 환경에서는 긴 FFmpeg 합성을 안정적으로 실행할 수 없어 `video-worker`를 별도 Docker 서비스에 배포합니다. Worker의 `README.md`와 `.env.example`을 따릅니다.

GY-NEXUS Vercel 환경변수:

```text
VIDEO_WORKER_URL=https://배포한-worker-주소
VIDEO_WORKER_SECRET=두-서비스에-같은-긴-비밀값
```

Worker 환경변수에는 같은 비밀값, `GY_APP_ORIGIN`, Supabase URL/Service Role Key가 필요합니다. BGM은 사용권을 가진 직접 파일 주소만 연결합니다.

## 저작권·계정 안전 규칙

- 권리 미확인 중국 영상은 키워드·훅·각도·리듬 분석 참고만 가능합니다.
- 좌우 반전, 속도 변경, 짧게 자르기는 저작권 해결 방법이 아닙니다.
- 최종 영상에는 직접 촬영, 판매자 제공, 제휴센터 제공, 사용 허가 확인 자료만 넣을 수 있습니다.
- 권리 미확인 소재를 최종 사용으로 선택하면 Runway, 최종 렌더, 게시가 차단됩니다.
- 보호된 도우인·샤오홍슈 영상을 서버가 임의 다운로드하지 않습니다. 장면 분석에는 대표님이 직접 만든 캡처와 메모를 사용합니다.

## 배포 확인

PowerShell 설치가 끝나면 프로젝트 폴더에서 다음을 실행합니다.

```powershell
git status
git add .
git commit -m "Add Shopping Shorts Center V2 Fast"
git push
```

GitHub와 Vercel이 연결되어 있다면 `git push` 뒤 자동 배포됩니다. Vercel 배포가 성공한 다음 새 쇼핑 쇼츠 주소를 새로고침합니다.
