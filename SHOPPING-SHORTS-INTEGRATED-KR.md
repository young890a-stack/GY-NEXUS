# GY-NEXUS 쇼핑 쇼츠 상품화 센터 통합판

## 이번 통합의 목표

상품 하나를 선택한 뒤 아래 흐름을 한 프로젝트에서 처리합니다.

1. 쿠팡·Temu 링크, 상품명, 검증된 설명, 사진 입력
2. 중국어 탐색 키워드와 도우인·샤오홍슈 검색 링크 생성
3. 참고 소재의 권리 상태 기록
4. 원본을 복제하지 않는 한국형 장면 순서와 훅 설계
5. 한국어 대본·AI 음성·정확한 SRT·썸네일 3안 생성
6. 생성 모델과 분리된 두 번째 AI가 허위·과장·제휴 고지·길이를 독립 검수
7. YouTube·Instagram·Douyin·Xiaohongshu 게시 패키지 생성
8. 대표 품질 승인 후에만 Runway와 게시 단계 허용

## 대표님이 사용하는 순서

`관리자 → Creative Studio Pro`로 이동합니다.

1. 직접 상품을 입력하거나 `인기상품·저장 상품 불러오기`에서 상품을 선택합니다.
2. 실제 상품 사진 1장 이상과 확인된 상품 설명을 입력해 프로젝트를 만듭니다.
3. 참고할 중국 영상이나 판매자 소재가 있으면 URL, 권리 상태, 분석 메모를 등록합니다.
4. `쇼핑 쇼츠 상품화 준비`를 누릅니다.
5. 중국어 탐색어와 검색 링크, 새로 설계한 컷, 플랫폼별 게시 패키지를 확인합니다.
6. 독립 광고 검수 결과가 통과인지 확인하고 첫 3초 훅 3개 중 하나를 선택합니다.
7. 저작권·상품 일치·허위 표현·자막·첫 3초를 확인하고 `선택한 훅으로 승인`을 누릅니다.
8. 이미지 품질검수를 통과한 뒤에만 Runway 비용을 승인합니다.
9. 완성 후 SRT·CapCut 안내서·게시 패키지·편집 JSON을 내려받습니다.
10. 최종 MP4가 있으면 YouTube 비공개 게시 대기열에 등록할 수 있습니다.

## 소재 권리 규칙

| 권리 상태 | 분석 참고 | 최종 영상 사용 |
|---|---:|---:|
| 직접 촬영 | 가능 | 가능 |
| 판매자 제공 | 가능 | 제공 조건 확인 후 가능 |
| 제휴센터 제공 | 가능 | 제휴센터 조건 확인 후 가능 |
| 사용 허가 확인 | 가능 | 가능 |
| 권리 미확인 | 가능 | 자동 차단 |

권리 미확인 자료에서 `최종 영상 사용`을 요청해도 서버가 자동으로 해제합니다. 비정상 요청으로 이 값을 우회해도 Runway 승인, 최종 MP4 렌더, 게시 대기열 등록 단계에서 다시 차단합니다. 최종 렌더 Worker에는 사용 허가 소재만 전달됩니다.

## 중국 플랫폼 연결 방식

이 통합판은 도우인·샤오홍슈의 보호된 페이지를 무단 수집하거나 영상을 내려받지 않습니다.

- 한국 상품명을 4~6개의 중국어 간체 탐색어로 변환합니다.
- 각 탐색어를 도우인·샤오홍슈 검색 화면으로 바로 여는 링크를 만듭니다.
- 참고 URL, 권리 상태, 대표님이 확인한 훅·각도·리듬 메모를 프로젝트에 저장합니다.
- AI는 상품 사실과 참고 메모를 바탕으로 0.7~2.5초의 새로운 컷 설계를 만듭니다.
- 실시간 인기 순위나 조회수를 확인한 것처럼 표시하지 않습니다.

로그인이 필요한 중국 영상을 실제로 시청했다고 가장하지 않는 것이 품질 원칙입니다. 향후 플랫폼의 승인된 API 또는 사용 허가 원본 파일 분석 Worker가 연결되면 같은 권리 구조 안에서 프레임 분석을 확장할 수 있습니다.

## 플랫폼별 결과물

- YouTube Shorts: 제목, 설명, 대본, 태그, 비공개 게시 대기열
- Instagram Reels: 캡션, 대본, 태그
- Douyin: 중국어 간체 제목, 캡션, 대본, 태그
- Xiaohongshu: 제목, 본문, 태그, 사진 노트 6~9장 구성
- 공통: GY 진열장 상품번호, 썸네일 3안, 제휴 고지, CTA
- 편집: 음성 대본과 글자가 일치하는 밀리초 SRT, CapCut 안내서, 편집 JSON

Instagram·Douyin·Xiaohongshu는 현재 계정 게시 권한이 연결되어 있지 않으므로 게시 패키지를 내려받아 각 공식 앱에서 최종 확인 후 올립니다. YouTube는 OAuth 연결 후 기존 통합 게시센터를 이용합니다.

## 정확한 SRT 처리

CapCut 자동 자막을 사용하지 않습니다. 쇼핑 대본을 그대로 보존한 뒤 문장 덩어리로 나누고 글자 수 비중에 따라 밀리초 타임코드를 배정합니다. SRT의 모든 문장을 합쳤을 때 AI 음성 대본과 글자가 다르면 콘텐츠 승인이 거부됩니다.

훅은 화면에 표시만 되는 선택지가 아닙니다. 대표님이 선택한 훅을 최종 대본 첫 문장에 결합한 뒤 SRT를 다시 만들며, 이 승인 이후에만 AI 음성 버튼이 열립니다.

## 비용 안전장치

- 상품화 준비: OpenAI 전략 모델만 사용
- 광고 품질검수: 생성 결과를 두 번째 독립 AI 호출로 다시 검사
- AI 음성: 별도 버튼을 눌렀을 때만 생성
- 이미지 생성: 별도 버튼을 눌렀을 때만 실행
- Runway: 이미지 품질 통과 + 콘텐츠 대표 승인 후에만 비용 승인 가능
- 최종 MP4: 기존과 같이 `VIDEO_WORKER_URL`이 연결되어야 자동 합성

Worker가 아직 없다면 Runway 장면 클립, SRT, CapCut 안내서, 게시 패키지, 편집 JSON까지 내려받아 CapCut에서 마무리할 수 있습니다.

## 쿠팡 API 최종 승인 전 운영

쿠팡 자동 수집을 미뤄도 제작은 멈추지 않습니다.

- 쿠팡 파트너스에서 직접 만든 제휴 링크를 입력합니다.
- Temu는 Affiliate 대시보드의 공유 링크를 사용합니다.
- 인기상품 소싱 센터에 저장된 승인 상품은 Creative Studio Pro에서 바로 불러옵니다.
- 제휴 링크가 없어도 콘텐츠를 만들 수 있지만 CTA에는 GY 상품번호를 사용합니다.

## 설치

1. 기존 프로젝트를 백업합니다.
2. 이 ZIP의 내용을 기존 GY-NEXUS 프로젝트 전체와 교체합니다.
3. Supabase SQL Editor에서 `supabase/SHOPPING-SHORTS-INTEGRATED-MIGRATION.sql`을 한 번 실행합니다.
4. 기존 환경변수에 `OPENAI_API_KEY`, Supabase 변수, Runway 변수가 연결되어 있는지 확인합니다.
5. 최종 MP4 자동 합성을 쓸 때만 `VIDEO_WORKER_URL`과 `VIDEO_WORKER_SECRET`을 연결합니다.
6. Vercel에 배포한 뒤 `관리자 → Creative Studio Pro`에서 새 테스트 프로젝트 1개를 만듭니다.

## 배포 후 1건 검수표

- 저장 상품 선택 시 상품명·설명·사진·제휴 링크가 채워지는가
- 상품화 준비 후 중국어 키워드와 두 플랫폼 검색 링크가 열리는가
- 권리 미확인 소재의 최종 사용 체크가 비활성화되는가
- 훅을 선택하지 않으면 대표 승인이 거부되는가
- SRT 문장이 음성 대본과 정확히 같은가
- 대표 승인 전 Runway 비용 승인 버튼이 잠겨 있는가
- 최종 MP4가 없으면 YouTube 게시 대기열 버튼이 보이지 않는가
- 완성본이 있어도 대표 승인 전 게시 대기열 등록이 서버에서 거부되는가
