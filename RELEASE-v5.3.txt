GY-NEXUS v5.3 PRODUCT INTELLIGENCE

핵심 추가 기능
1. 상품 URL 한 줄 입력
2. JSON-LD / Open Graph 기반 상품명, 설명, 이미지, 가격 자동 추출
3. OpenAI 기반 상품 특징, 장점, 타깃 고객, 사용 상황, 판매 포인트 분석
4. SEO 키워드, 쇼츠 훅, 콘텐츠 방향 자동 생성
5. 분석 결과 Supabase trend_products 저장
6. 자동 추출 실패 시 수동 보완 입력 제공
7. 정식 상품 전환 및 AI 콘텐츠 제작 화면 연결

실행 전 확인
- .env.local에 Supabase와 OpenAI 환경변수 입력
- 기존 v5.2 schema.sql이 실행되어 trend_products 테이블이 존재해야 함
- npm install
- npm run dev

주의
- 일부 쇼핑몰은 서버 접근을 차단하거나 로그인/동적 렌더링을 사용하여 자동 추출이 제한될 수 있음
- 그런 경우 수동 보완 입력에 상품명, 가격, 설명을 입력하면 AI 분석은 정상 진행됨
