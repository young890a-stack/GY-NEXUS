# GY-NEXUS 사진 한 장 쇼츠 상품화 센터

## 이번에 추가된 기능

- 쿠팡·Temu 상품 사진 **1장만으로 시작하는 쇼츠 모드**
- 15초·20초·25초·30초 길이 선택
- 한 장 모드와 프리미엄 2~4장 모드 분리
- 상품명과 검증된 설명을 바탕으로 한국형 판매 패키지 생성
  - 첫 2초 훅 3개
  - 전체 한국어 내레이션 대본
  - 제목·설명·해시태그
  - 제휴 고지와 프로필 링크 CTA
  - 썸네일 문구 3안
- 정확한 대본으로 OpenAI 한국어 AI 음성 생성·미리듣기
- 강조형·카드형·미니멀 자막 스타일 저장
- 썸네일 혜택 강조·문제 해결·상품 중심 스타일 저장
- 장면별 효과음 권장 수준 저장
- 게시 패키지 TXT, SRT, CapCut 안내서, 편집 JSON 내보내기
- 사진 한 장에서는 보이지 않는 뒤·옆면·버튼·포트를 새로 만들지 않도록 생성·검수 제한
- 이미지 생성과 Runway 비용은 기존과 같이 대표님이 별도로 승인

## 처음 한 번만 적용

### 1. Supabase SQL

Supabase `SQL Editor`에서 아래 파일 전체를 실행합니다.

`supabase/PHOTO-COMMERCE-SHORTS-MIGRATION.sql`

기존 데이터를 삭제하지 않고 Creative Studio Pro에 15초 영상 길이만 추가합니다.

### 2. 프로젝트 교체와 배포

교체용 ZIP을 기존 GY-NEXUS 폴더에 전체 덮어쓴 뒤 PowerShell에서 실행합니다.

```powershell
npm ci
npm run build
git add .
git commit -m "Add one-image photo commerce shorts studio"
git push origin main
```

`nothing to commit`이 나오면 `git push origin main`만 실행합니다.

## 실제 사용 순서

1. `AI Factory → Creative Studio Pro`를 엽니다.
2. `사진 한 장 쇼츠`를 선택합니다.
3. 상품명, 검증된 상품 설명과 직접 만든 제휴 링크를 입력합니다.
4. 정면이 선명한 상품 사진 1장을 올립니다.
5. 15~30초 길이, 자막·썸네일·효과음 스타일을 선택합니다.
6. `장면 기획 프로젝트 만들기`를 누릅니다.
7. `판매 패키지 생성`을 눌러 훅·대본·썸네일·제목·설명·태그를 확인합니다.
8. 대본이 마음에 들면 `AI 음성 만들기`를 누르고 미리 듣습니다.
9. `전체 이미지 검수`를 누릅니다.
10. 모든 장면이 통과했을 때만 `Runway 비용 승인`을 누릅니다.
11. 장면 영상을 만들고 최종 MP4 합성 또는 CapCut 자료를 내려받습니다.

## 사진 한 장 모드의 품질 원칙

- 원본 사진에서 확인되지 않는 상품 뒷면이나 측면을 만들지 않습니다.
- 실제 상품 형태와 색상은 사실 기준으로 고정합니다.
- 배경, 조명, 구도와 그래픽 연출만 바꿉니다.
- 상품을 직접 사용하지 않았다면 후기처럼 표현하지 않습니다.
- AI가 만든 장면은 상품 일치도·형태·색상·로고·글자·영상 안정성 검수를 통과해야 합니다.
- 권리 확인이 되지 않은 도우인·샤오홍슈 영상은 최종 영상 소재로 사용하지 않습니다.

## 환경변수

기존 `OPENAI_API_KEY`, `OPENAI_IMAGE_MODEL`, `RUNWAYML_API_SECRET`, Supabase 설정을 사용합니다.

아래 값은 선택사항이며 입력하지 않아도 기본값으로 작동합니다.

```env
OPENAI_TTS_MODEL=gpt-4o-mini-tts
OPENAI_TTS_VOICE_FEMALE=marin
OPENAI_TTS_VOICE_MALE=cedar
```

## 비용 안전장치

- 프로젝트 생성: 장면 기획만 수행
- 판매 패키지 생성: 텍스트 AI 호출
- AI 음성 만들기: 버튼을 누른 경우에만 음성 호출
- 이미지 생성: 기존 별도 버튼
- Runway 영상: 모든 이미지 통과 후 대표님 명시 승인

외부 AI 실사용 호출은 API 비용이 발생하므로 자동 빌드 검증에서는 실행하지 않았습니다.
