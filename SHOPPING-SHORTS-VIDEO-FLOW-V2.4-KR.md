# GY-NEXUS 쇼핑 쇼츠 영상 흐름 통합 V2.4

## 영상 전체 확인 결과

대표님이 보내주신 4분 45초 영상의 실제 제작 흐름을 다음과 같이 반영했습니다.

```text
상품·키워드 입력
→ 중국 소스 검색
→ 검색 영상 직접 재생
→ 여러 영상 선택
→ 목표 길이·속도·자막·음성 설정
→ AI 상품·장면 분석
→ 0.7~2.5초 컷 재구성
→ 중국어 음성 제거·한국어 대본/TTS
→ 정확한 한국어 자막
→ 첫 3초 훅 생성·승인
→ 썸네일 3안
→ 제목·설명·태그·제휴 고지
→ CapCut 자료·MP4·게시 대기열
```

## V2.4 핵심 업데이트

- 검색 카드마다 `영상 재생` 버튼을 추가했습니다.
- 도우인은 공개 VideoID가 있으면 도우인 공식 IFrame 플레이어로 GY-NEXUS 안에서 재생합니다.
- 공식 재생이 불가능한 도우인 주소와 샤오홍슈는 로그인된 Edge 원문 플레이어에서 즉시 재생합니다.
- 미리보기 화면에서 바로 AI 믹스 선택·해제를 할 수 있습니다.
- 선택한 여러 영상을 한 버튼으로 소스 저장 → 편집 설정 → AI 컷 → 한국어 판매 패키지까지 연결합니다.
- 영상 속 1.4배속 기능을 참고해 1.0x, 1.2x, 1.4x 편집 속도를 추가했습니다.
- 중국어 화면 자막은 기본적으로 원본을 복제하지 않고 새 장면으로 재제작해 제거합니다.
- 허가 영상에는 하단 안전 크롭 또는 원문 유지 방식을 선택할 수 있습니다.
- 원본 중국어 음성은 제거하고 검수된 한국어 AI 음성 또는 무음으로 처리합니다.
- 컷 설계, 한국어 훅·대본·정확한 SRT, AI 음성, 썸네일 3안, 게시 정보, 내보내기를 기존 기능과 연결했습니다.
- 편집 속도와 자막·음성 처리 설정이 FFmpeg Worker와 CapCut 내보내기 자료에 포함됩니다.

## 실제 사용 순서

1. `/admin/shopping-shorts`에서 프로젝트를 선택합니다.
2. 한국어 상품명을 입력하고 `중국어로 자동 번역해 찾기`를 누릅니다.
3. 카드의 `영상 재생`을 눌러 실제 내용을 확인합니다.
4. 필요한 쇼츠를 여러 개 선택합니다.
5. 편집 속도, 중국어 화면 자막, 원본 음성 처리 방식을 고릅니다.
6. `선택 N개로 쇼핑 쇼츠 자동 제작 준비`를 누릅니다.
7. AI 컷 타임라인과 한국어 훅 3개를 확인합니다.
8. 훅 하나를 선택하고 대표 품질 승인을 누릅니다.
9. `AI 음성 만들기`로 한국어 TTS를 생성합니다.
10. 상품 이미지와 장면 품질을 통과시킨 뒤 Runway 비용을 승인합니다.
11. 장면 영상 생성 후 FFmpeg Worker로 최종 MP4를 합성합니다.
12. SRT, CapCut JSON, 썸네일 SVG, 게시 패키지를 내려받거나 YouTube 비공개 게시 대기열로 보냅니다.

## 재생 방식과 권리 규칙

- 도우인 사이트 내 재생은 공식 공개 임베드 API만 사용합니다.
- 샤오홍슈에는 동일한 공개 임베드 API가 확인되지 않아 로그인된 원문 플레이어를 사용합니다.
- 검색 영상은 훅, 촬영각도, 상품 설명 구조 분석용입니다.
- 검색 결과 원본을 자동 다운로드하거나 워터마크를 제거하지 않습니다.
- 직접 촬영, 판매자 제공, 제휴센터 제공 또는 사용 허가 영상만 최종 원본 컷으로 사용할 수 있습니다.
- 권리 미확인 링크는 AI가 새로운 한국형 장면으로 재설계합니다.
