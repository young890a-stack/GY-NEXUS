# GY-NEXUS 중국 쇼츠 Edge 계정 연결 도우미

이 확장 프로그램은 대표님이 Edge에서 이미 로그인한 도우인·샤오홍슈 검색 화면을 사용합니다.

## 하는 일

- GY-NEXUS에서 만든 중국어 검색어로 도우인·샤오홍슈 검색 탭을 백그라운드에서 엽니다.
- 검색 화면에 실제로 표시된 개별 쇼츠 링크, 썸네일, 공개 좋아요 문구, 길이, 해시태그를 최대 12개 읽습니다.
- 60초를 넘는 것으로 확인된 영상은 제외합니다.
- 수집한 카드와 반복 태그를 GY-NEXUS 쇼핑 쇼츠 센터로 돌려보냅니다.

## 하지 않는 일

- 도우인·샤오홍슈 비밀번호를 받지 않습니다.
- 쿠키 권한을 요청하지 않고 쿠키를 읽거나 서버로 전송하지 않습니다.
- 비공개 API를 호출하지 않습니다.
- 영상 파일 주소를 추출하거나 원본 영상을 다운로드하지 않습니다.
- 워터마크 제거 또는 무단 재배포를 하지 않습니다.

## Edge 설치

1. Edge 주소창에 `edge://extensions`를 입력합니다.
2. 왼쪽 아래 `개발자 모드`를 켭니다.
3. `압축 풀린 항목 로드`를 누릅니다.
4. 이 `gy-china-account-connector` 폴더를 선택합니다.
5. 도우인과 샤오홍슈에 각각 로그인해 둡니다.
6. GY-NEXUS의 `/admin/shopping-shorts`를 새로고침합니다.
7. `Edge 로그인 계정 연결 · 연결됨`이 표시되면 준비 완료입니다.

## 사용

GY-NEXUS에 한국어 상품명을 입력하고 `한국어로 인기 쇼츠 찾기`를 누릅니다. 공개 웹 검색 결과가 먼저 표시되고, 연결 도우미가 설치되어 있으면 로그인 계정의 실제 검색 카드와 반복 해시태그가 이어서 합쳐집니다.

플랫폼에서 로그인 확인이나 보안 문자가 나오면 자동 수집을 중지하고 해당 탭을 화면에 띄웁니다. 로그인이나 확인을 직접 마친 뒤 GY-NEXUS에서 `로그인 계정으로 다시 찾기`를 누르세요.
