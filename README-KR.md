# GY-NEXUS 핵심 4개 최상단 배치 패치 V2

이 패치는 메뉴를 삭제하지 않습니다.

## 최상단 핵심 4개

1. 쇼핑 쇼츠 제작실
2. 모바일 원클릭 쇼츠
3. 블로그 제작실
4. 상품 관리

## 아래쪽에 그대로 보존되는 메뉴

기존 운영·분석·콘텐츠·자동화·게시·연결·시스템 메뉴는 모두 아래쪽에 남습니다.
그룹 이름에 `단계별 보완`이 표시되어, 당장 사용할 메뉴와 나중에 개선할 메뉴를 쉽게 구분할 수 있습니다.

최근 추가한 쇼츠 기능, 기존 관리자 페이지, 데이터베이스, 환경변수는 삭제하지 않습니다.

## 모바일 하단

`홈 · 쇼츠 · 원클릭 · 블로그 · 상품`으로 구성됩니다.

## 적용 방법

1. ZIP 압축을 풉니다.
2. 압축을 푼 폴더를 GY-NEXUS 프로젝트 폴더 안에 넣습니다.
3. 프로젝트 폴더에서 `1-핵심4개최상단적용.bat`를 실행합니다.
4. 자동 백업 후 파일 2개를 교체하고 `npm run build`를 실행합니다.
5. 빌드 성공 후 GitHub Desktop에서 Commit 후 Push합니다.

추천 커밋 문구:

`Put core creator tools at the top of admin navigation`
